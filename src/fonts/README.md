# Шрифты

Лежат прямо в репозитории намеренно. Требование плана (раздел 3.2): ни одного
внешнего запроса при загрузке страницы. Российские провайдеры рвут трафик к
зарубежным CDN, поэтому `fonts.googleapis.com` и любой другой внешний источник
шрифта запрещены.

Файлы вынуты из npm-пакетов `@fontsource-variable/*` (версия 5.3.0), сами пакеты
после этого удалены из зависимостей - они больше не нужны, а неиспользуемая
зависимость это мусор в проекте.

| Файл | Гарнитура | Где применяется |
|---|---|---|
| `unbounded-*` | Unbounded Variable, 200-900 | заголовки, крупные цифры кейсов |
| `onest-*` | Onest Variable, 100-900 | основной текст |
| `jetbrains-mono-*` | JetBrains Mono Variable, 100-800 | надзаголовки, цены, время, счётчик баллов |

Взяты только два подмножества из каждой гарнитуры: `cyrillic` и `latin`.
Кириллица проверена - это глифы самого шрифта, а не подстановка системным.
Латиница нужна для цифр, знака `+` и латинских вкраплений.

Лицензия у всех трёх - SIL Open Font License 1.1, использование на сайте
разрешено, в том числе коммерческое.

Обновление: поставить нужный `@fontsource-variable/<имя>`, скопировать
`files/<имя>-cyrillic-wght-normal.woff2` и `files/<имя>-latin-wght-normal.woff2`
сюда, пакет удалить.
